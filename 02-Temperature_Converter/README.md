# 🌡️ Temperature Converter

A simple command-line temperature converter built in Python that converts between Celsius and Fahrenheit.

## 🚀 How It Works

1. Enter the temperature
2. Enter the unit (C or F)
3. Get the converted result instantly

## 💻 Usage

```bash
python Temperature_Converter.py
```

## 🛠️ Tech Stack

- Python 3.14

## 📌 What I Learned

- String methods with `.strip()` and `.upper()`
- Type conversion with `float()`
- Formatted output with f-strings and `round()`
