# ⏳ Countdown Timer

A simple command-line countdown timer built in Python. Enter the number of seconds, and watch the timer count down in `HH:MM:SS` format until it hits zero.

## 📸 Demo

```
Enter the number of seconds: 10
00:00:10
00:00:09
00:00:08
...
00:00:00
Time's Up
```

## 🚀 Features

- Takes user input in seconds
- Converts seconds into `hours:minutes:seconds` format
- Live updating countdown display (updates every second)
- Clean and beginner-friendly code

## 🛠️ Tech Stack

- Python 3
- Built-in `time` module

## 📂 How to Run

1. Clone the repository
   ```bash
   git clone https://github.com/DevanshKumar01/Countdown-Timer-Program.git
   cd Countdown-Timer-Program
   ```

2. Run the script
   ```bash
   python Countdown_Timer-Program.py
   ```

3. Enter the number of seconds when prompted, and watch it count down!

## 🧠 What I Learned

- Using `reversed(range())` to loop backwards
- Formatting numbers with `:02` to pad with leading zeros
- Using `time.sleep()` to create a delay between iterations
- Basic integer division (`//`) and modulo (`%`) for time conversion logic
