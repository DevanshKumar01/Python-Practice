
# ⚖️ Weight Converter

A simple command-line weight converter built in Python that converts between kilograms and pounds.

## 🚀 How It Works

1. Enter your weight
2. Enter the unit (Kg or Lb)
3. Get the converted result instantly

## 💻 Usage

```bash
python weight_converter.py
```

## 🛠️ Tech Stack

- Python 3.14

## 📌 What I Learned

- String methods with `.strip()` and `.lower()`
- Type conversion with `float()`
- Formatted output with f-strings and `round()`
